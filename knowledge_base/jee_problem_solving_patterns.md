# JEE Problem Solving Patterns

## A general workflow
1. Parse the statement and define variables.
2. Identify topic and matching formula family.
3. Apply transformations (factorization, substitution, simplification).
4. Verify domain, constraints, and edge cases.
5. Present final answer with short justification.

## Heuristics
- If expression is polynomial-like, try factorization first.
- If constraint-heavy, use case analysis.
- If probability has "at least" or "at most", test complement method.
- If word problem cannot be symbolized directly, write intermediate equations.

## Pitfalls
- Jumping to computation before fully modeling the question.
- Stopping after first root without checking all valid roots.
- Ignoring question asks (value, count, set, interval).
