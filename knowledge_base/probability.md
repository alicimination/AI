# Probability Basics (JEE)

## Core Formulas
- `P(A) = favorable outcomes / total outcomes` (equally likely case).
- `P(A') = 1 - P(A)`.
- `P(A union B) = P(A)+P(B)-P(A intersection B)`.

## Typical Approach
1. Build sample space.
2. Count favorable outcomes.
3. Validate probability in `[0,1]`.

## Common Mistakes
- Double counting events.
- Invalid sample space assumptions.
