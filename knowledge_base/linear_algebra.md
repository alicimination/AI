# Linear Algebra Basics (JEE)

## Core Facts
- Determinant of 2x2 matrix `[[a,b],[c,d]]` is `ad-bc`.
- Invertible iff determinant is non-zero.
- Matrix multiplication is associative, not commutative.

## Typical Approach
1. Verify matrix dimensions.
2. Compute determinant/rank where needed.
3. Check singular/non-singular cases.

## Common Mistakes
- Multiplication order mismatch.
- Arithmetic errors in cofactors.
