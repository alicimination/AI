# Trigonometry Basics for Mixed Problems

## Standard identities
- sin^2 x + cos^2 x = 1
- 1 + tan^2 x = sec^2 x
- 1 + cot^2 x = csc^2 x

## Angle formulas
- sin(A ± B) = sinA cosB ± cosA sinB
- cos(A ± B) = cosA cosB ∓ sinA sinB

## Strategy notes
- Convert everything to sin/cos when expressions are mixed.
- For equations, restrict solutions to stated domain.
- Use principal values carefully for inverse trigonometry.

## Pitfalls
- Forgetting periodic general solutions.
- Squaring equations introduces extraneous roots.
