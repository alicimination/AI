# Common Pitfalls and Constraints

## Domain Constraints
- `log(x)` requires `x > 0`.
- `sqrt(x)` for real output requires `x >= 0`.
- Rational expression denominator cannot be zero.

## Exam Mistakes
- Premature cancellation that removes valid/invalid roots.
- Failing to test extraneous roots.
- Probability outputs outside `[0,1]`.

## Verification Checklist
1. Substitute result into original equation.
2. Check domain and inequality constraints.
3. Evaluate edge cases or boundary conditions.
