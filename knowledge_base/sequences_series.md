# Sequences and Series (JEE Basics)

## Arithmetic progression (AP)
- nth term: a_n = a + (n-1)d
- Sum of first n terms: S_n = n/2 * [2a + (n-1)d]

## Geometric progression (GP)
- nth term: a_n = ar^(n-1)
- Sum of first n terms: S_n = a(1-r^n)/(1-r), r != 1
- Infinite GP sum (|r| < 1): S_inf = a/(1-r)

## Useful approach
- Convert verbal condition into equations in a, d, r.
- Keep track of index carefully (first term is n=1).

## Pitfalls
- Sign mistakes when r is negative.
- Using infinite GP formula when |r| >= 1.
- Off-by-one errors in nth term indexing.
